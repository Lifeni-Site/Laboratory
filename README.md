# Laboratory

一个实验室，在这里做一些有趣的东西。

大概在 [https://lab.lifeni.life](https://lab.lifeni.life) 。

## 技术

使用 Solid 和 Windi CSS 构建页面，部署在 Netlify 上。

使用 pnpm 作为包管理工具，Snowpack 作为构建工具。
